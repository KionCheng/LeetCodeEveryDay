## 数据结构的**存储**方式

|            | 数组                                          | 链表                                            |
| ---------- | --------------------------------------------- | ----------------------------------------------- |
| 存储       | 一整块空间，连续储存                          | 非连续储存                                      |
| 访问       | 随机访问（通过索引访问）                      | 不能随机访问                                    |
| 空间利用率 | 好                                            | 不好（需要存储前后指针）                        |
| 扩容       | 需要一整块更大空间，集体复制，时间复杂度O(N)  | 不存在扩容问题                                  |
| 增删       | 搬移后面的所有数据以保持连续，时间复杂度 O(N) | 已知前驱和后驱，操作指针即可，时间复杂度 O(1)。 |
